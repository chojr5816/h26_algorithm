문제 18번
0. Cart 테이블 생성 (DDL)

CREATE TABLE Cart (
    cartid   NUMBER PRIMARY KEY,
    custid   NUMBER,
    bookid   NUMBER,
    cartdate DATE,
    FOREIGN KEY (custid) REFERENCES Customer(custid),
    FOREIGN KEY (bookid) REFERENCES Book(bookid)
);


문제 18 조회 질의 (SQL)
(1) 고객번호 1번의 cart에 저장된 도서 중 이미 주문한 도서를 구하시오.
SQL : 
SELECT bookid FROM Cart WHERE custid = 1
INTERSECT
SELECT bookid FROM Orders WHERE custid = 1;

결과 : 
BOOKID
----------
         1
(2) 고객번호 1번의 cart에 저장된 도서 중 아직 주문하지 않은 도서를 구하시오.
SQL : 
SELECT bookid FROM Cart WHERE custid = 1
MINUS
SELECT bookid FROM Orders WHERE custid = 1;

결과 : 
    BOOKID
----------
         2

(3) 고객번호 1번의 cart에 저장된 도서들의 정가 합계를 구하시오.
SQL : 
SELECT SUM(b.price) AS total_cart_price
FROM Cart c
JOIN Book b ON c.bookid = b.bookid
WHERE c.custid = 1;

결과 : 
TOTAL_CART_PRICE
----------------
           60000
